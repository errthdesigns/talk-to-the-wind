# Talk to the Wind

An interactive particle system that responds to your emotions. Type how you feel, and watch the wind transform into shapes that reflect your words.

## Features

- **30 unique responses** - The wind recognizes specific phrases and responds with poetic messages
- **16 particle shapes** - Heart, galaxy, tornado, rain, flame, bird, tree, mountain, and more
- **9 mood states** - Love, peaceful, fierce, melancholy, joyful, mysterious, curious, playful, calm
- **7 motion patterns** - Drift, swirl, gust, breathe, tremble, pulse, orbit
- **60,000 glowing particles** with multi-layer glow effects
- **Synth-inspired UI** - Minimal, technical aesthetic with sliders and system logs

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- **React** - UI framework
- **Three.js** - 3D particle rendering
- **Vite** - Build tool
- **Tailwind CSS** - Styling

## Controls

- **TUR** (Turbulence) - How chaotic the particle movement is
- **VEL** (Velocity) - Speed of particle motion  
- **RES** (Resonance) - Connection strength between particles
